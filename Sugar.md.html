Sugar

Free Features:

  • s!donate <time> | <amount/what> | <#ofwinners> | <message> sends a donation request from you to a mod. (Requires Cinnamon)
  • s!example sends an example on how to donate.
  • s!setlogchannel <channelid> sets the log channel for the giveaway donations. (Mods Only)
  • s!invite gives you an invite link for Cinnamon and Sugar.
  • s!partners Displays Sugar’s partners.
  • s!ping tells you bot’s ping time
  • s!pro tells you if your server is a pro server
  • s!help this message

Pro Features

  • s!setpingrole <roleid> sets the role that gets pinged when there is a donation. (Mods Only)
  • s!setdonationchannel <channelid> sets the channel where you can donate, locks donation in other channels.

Permissions Required:

  • Send Messages Because, well, duh
  • Embed Links Some bot embeds need this permission
  • Read Message History Inline replies need this permission
  • Mention @Here etc Giveaway Management Pings
  • Add reactions So you know it hears you
